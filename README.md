# Playwright Boilerplate
    
Playwright end-to-end test automation boilerplate, Using [ES Modules](https://hacks.mozilla.org/2018/03/es-modules-a-cartoon-deep-dive/), [Babel](https://babeljs.io), [Jest](https://jestjs.io) and Expect.
    
## Getting Started
    	1. git clone git@github.com:e2e-boilerplate/playwright-es-modules-babel-jest-expect.git
    	2. cd playwright-es-modules-babel-jest-expect
    	3. npm install
    	4. npm run test
        
    
For more boilerplate click [here](https://github.com/e2e-boilerplate/utils/blob/master/docs/implemented.md)
